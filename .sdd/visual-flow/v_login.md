# ログイン

## パス

`/login`

## 概要

一般ユーザー・組織管理者のログイン画面。メールアドレスのみでログイン（パスワードなし）。

## 遷移元・遷移先

[transition-flow.md](transition-flow.md) を参照。

- 遷移先: 所属組織 1 件 → /projects、複数件 → /select-org
- 管理者としてログイン時 → /admin にアクセス可能

## UI 要素

- ロゴ（FolderKanban アイコン + 「ProjectHub」）
- 見出し「ログイン」
- メールアドレス入力（必須）
- 「管理者としてログイン」チェックボックス
- ログインボタン
- スーパー管理者ログインへのリンク（/super-admin/login）

## 備考

- 未登録メールの場合は `POST /users` でユーザー自動作成後にログイン
- 組織管理者または is_admin ユーザーのみ管理画面にアクセス可能
