# 管理画面

## パス

`/admin`

## 概要

組織管理者向けの管理画面トップ。役職管理（/admin/roles）にリダイレクトする。

## 遷移元・遷移先

[transition-flow.md](transition-flow.md) を参照。

- 遷移元: /projects（ヘッダーの管理画面リンク）
- 遷移先: 自動で /admin/roles にリダイレクト

## UI 要素

- サイドナビ: プロジェクト管理、部署管理、役職管理、ステータス管理、ユーザー管理、ワークフロー、Issueテンプレート
- ヘッダー: 「管理画面」、戻るリンク（/projects）

## 備考

- 組織管理者または is_admin ユーザーのみアクセス可能
- ログイン時に「管理者としてログイン」チェックが必要
