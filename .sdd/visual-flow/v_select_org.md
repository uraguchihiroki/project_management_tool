# 組織を選択

## パス

`/select-org`

## 概要

所属組織が複数ある場合に、ログインする組織を選択する画面。

## 遷移元・遷移先

[transition-flow.md](transition-flow.md) を参照。

- 遷移元: /login（組織が複数件の場合）
- 遷移先: 組織選択後 → /projects

## UI 要素

- 見出し「組織を選択」
- 説明文「ログインする組織を選択してください」
- 組織カード一覧（クリックで選択）
- 所属組織がない場合のメッセージ

## 備考

- `GET /users/:id/organizations` で所属組織一覧取得
- 組織 1 件の場合は自動で /projects にリダイレクト
